# MaintenAI 🚀
AI-Powered Predictive Maintenance Platform

## 🎯 Overview
MaintenAI is a web-based platform that predicts machine failures using AI-based anomaly detection.

## 💡 Problem
Industrial machines often fail unexpectedly due to lack of real-time monitoring and predictive maintenance systems.

## 🚀 Solution
MaintenAI analyzes sensor data (temperature, vibration, pressure) to:
- detect anomalies
- calculate health score
- provide early warning alerts

## 🧠 AI Approach
Using Isolation Forest (unsupervised learning) to detect abnormal patterns in machine data.

## 📊 Features
- Real-time monitoring dashboard
- AI anomaly detection
- Machine health score
- Early warning system
